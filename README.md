# EC1 Gallery
This is a responsive, static website showcasing a design concept for EC1 Gallery. It uses jQuery (for javascript), Bootstrap 4 (for styling) and Liveicons (for icons).

It is also hosted on https://ec1-gallery.bitballoon.com/

## Installation
1. Clone this repo (or unzip the zip file).
2. Change the directory to the root of this folder.
2. Open index.html in your browser.

## Functionality Implemented
1. Fully responsive website
2. Pure css implementation of hamburger pull-out menu/sidebar
3. Sidebar menu indicates active link based on section active in viewport.
4. Simple css transition (to pull out the search bar) on search icon
